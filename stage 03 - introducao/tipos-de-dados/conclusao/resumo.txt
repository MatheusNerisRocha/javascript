Tipos de dados

Conforme o ECMAScript standard temos 9 tipos de dados:

* Data types
    * Primitive / Primitive value
    * Structural
    * Structural Primitive

Primitivos

* String -> "ABC" "BCA" "bca" "bCa"
* Number -> 23 32 24
* Boolean -> true ou false
* undefined
* Symbol
* BigInt

Estruturais
    * Object
        * Array
        * Map
        * Set
        * Date
        * ...

* Function

Primitivo Estrutural / Structural Root Primitive

* null
